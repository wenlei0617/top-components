# Icon

语义化的矢量图形。

## 使用说明

<Playground>
  <CheckBox name="loading" checked>

</Playground>

## API

| 属性 | 说明     | 类型    | default |
| ---- | -------- | ------- | ------- |
| name | 图标类型 | string  | -       |
| fill | 填充色   | string  | "#000"  |
| spin | 是否旋转 | boolean | false   |
