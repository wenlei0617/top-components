### 进度条 api

| 属性        | 说明         | 类型   | default |
| ----------- | ------------ | ------ | ------- |
| strokeWidth | 边框粗细     | number | 2       |
| height      | 圆的高度     | number | 120     |
| width       | 圆的宽度     | number | 120     |
| rate        | 当前进度     | number | 20      |
| color       | 进度条颜色   | string | "red"   |
| backColor   | 进度条背景色 | string | "grey"  |
